# Responsive Glass Sidebar

### Responsive Glass Sidebar

- Responsive Glass Sidebar Using HTML CSS JavaScript 
- It contains a beautiful background image.
- It has a glass effect on the side bar.
- It contains a Dropdown menu that opens and closes.
- Developed first with the Mobile First methodology, then for desktop.
- Compatible with all mobile devices and with a beautiful and pleasant user interface.
